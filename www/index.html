BlockPlot is a web app where you can create new 3D worlds or upload existing worlds from other popular voxel apps and share them online with your friends.

It is made by the team that created Voxel.js, the most advancedÂ open source JavaScript technology for rendering voxel worlds in web browsers.

Requirements

BlockPlot requires a web browser that supports WebGL. It will work on most computers that can play other 3D video games.

You can find out if your browser supports WebGL by visiting this test page.
